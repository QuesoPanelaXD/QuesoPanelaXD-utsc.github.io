Mis Proyectos

Proyecto 1 – Espacios Seguros

Desarrollé una aplicación que mostraba en un mapa diferentes lugares tranquilos y accesibles para personas con autismo. El objetivo era identificar sitios con bajo nivel de ruido donde pudieran realizar sus actividades cotidianas de forma cómoda y sin distracciones sensoriales.

Proyecto 2 – Semáforo de Ruido

Trabajé en una aplicación orientada a personas con autismo, diseñada como un “semáforo” para medir y señalar el nivel de ruido en distintos lugares. El sistema mostraba tres niveles: rojo para ambientes muy ruidosos con advertencia, amarillo para ruido moderado y verde para espacios silenciosos y seguros.

Proyecto 3 – Plataforma de Capacitación (Empresa Álcali)

Colaboré en el diseño de una plataforma web destinada a la capacitación de nuevos trabajadores. El sistema permitía a los empleados realizar exámenes en línea y subir actividades de sus cursos de inducción, facilitando la organización y seguimiento del proceso formativo.

Proyecto 4 – Sistema de Citas

Participé en el desarrollo de una página web que permitía a los usuarios registrarse, agendar citas y cancelarlas de manera sencilla. Este proyecto buscaba optimizar la gestión de horarios y mejorar la comunicación entre clientes y proveedores de servicios.

Proyecto 5 – Registro Escolar

Diseñé un sistema para la gestión de estudiantes, donde se podían registrar datos académicos y administrativos. La plataforma facilitaba el control de información y mejoraba la organización en el entorno escolar.

Proyecto 6 – Aplicación Escolar

Desarrollamos una aplicación enfocada en el registro y administración de datos de una institución educativa. Permitía consultar información de alumnos, organizar materias y agilizar el manejo académico en dispositivos móviles.
